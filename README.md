## ⚙️ How to Run the Application on Windows  

This app uses Node.js, React, and MongoDB. To keep things simple, this version comes with a ready setup to start the app without manual coding. You only need to run two files that will start the backend (server) and frontend (what you see) for you.  

### Step 1: Open the App Folder  

Go to the folder where you extracted the files.  

### Step 2: Start the Backend (Server)  

- Inside the folder, look for a file named `start-backend.bat`.  
- Double-click that file.  

This will open a new window. It runs the backend server, which talks to the database and manages the app logic.  

### Step 3: Start the Frontend (User Interface)  

- Go back to the main folder.  
- Find `start-frontend.bat`.  
- Double-click it.  

Another window will open and start the frontend. This displays the app in your browser.  

### Step 4: Open the App in Your Browser  

Once both windows are running, open your preferred web browser (like Chrome, Edge, or Firefox).  

Type this address in the address bar:  

`http://localhost:3000`  

Press Enter. You should see the app’s homepage appear.

## 🧾 Troubleshooting Tips  

- If you get an error when running the `.bat` files, make sure your folder contains all files and none are missing.  
- Ensure no other app is using port 3000 on your computer. If it is, close that app and try again.  
- If the page does not load, check your internet connection.  
- You can restart the app anytime by repeating the start steps.  

## 🚩 About this Repository  

This project uses these main technologies:  

- axios for making network requests  
- bcryptjs for password security  
- crypto for secure processes  
- expressjs for the backend server  
- jsonwebtoken (JWT) for authentication  
- mongodb and mongoose for database  
- multer for uploading files  
- nodejs for running JavaScript on the server  
- react, react-redux, react-router for the frontend  
- redux-toolkit for state management  
- tailwindcss for styling the user interface  

These combine to provide a fast, secure, and fully functional online store experience.
